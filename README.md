🛠️ Tech Stack


Python
Pandas / NumPy — data manipulation
Matplotlib / Seaborn — data visualization
Scikit-learn — model building and evaluation


📊 Project Workflow

Each notebook follows a similar structure:


Data Loading & Exploration — initial inspection of the dataset, checking for nulls, data types, and summary statistics
Data Preprocessing — handling missing values, encoding categorical variables, feature scaling where needed
Exploratory Data Analysis (EDA) — visualizing relationships between features and the target variable
Model Building — training a Logistic/Linear Regression model on the processed data
Model Evaluation — assessing performance using relevant metrics (accuracy, precision/recall for classification; R², MAE/RMSE for regression)
